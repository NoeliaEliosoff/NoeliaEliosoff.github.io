Design Principles

Visual Hierarchy

Company Name: Freepik

www.freepik.es

Visual Hierarchy is exemplified in this site because the order in which things are presented, is the one which the user would usually do. It makes important things and links prominent, for exmple the purpose of the page is in a bigger sizethan the rest, and the button to start looking for content is in a different color.

Hick's Law

Company Name: Healthline

www.healthline.com

This is a good example of Hick's Law because you don't have many options to choose, so it's more likely that the user is quickly going to find what he needs.

White Space - Clean Design

Company Name: Flaticon

www.flaticon.com

This site makesgreat use of the space, and communicates effectively. The user can find what he needs, but there aren't too many things on the screen.
